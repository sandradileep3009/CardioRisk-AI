# ❤️ CardioRisk AI — Clinical Cardiovascular Risk Stratification Platform

## Overview

CardioRisk AI is an explainable machine learning-based cardiovascular risk assessment platform designed to estimate a patient's **10-year cardiovascular disease risk** using clinical and lifestyle factors.

The project upgrades traditional heart disease prediction into a **clinical decision-support prototype** by combining:

- Machine learning risk prediction
- Probability calibration
- Risk stratification
- Confidence interval estimation
- Model explainability
- Interactive Streamlit deployment


> ⚠️ This project is an AI research prototype and is not a medical diagnosis system.
